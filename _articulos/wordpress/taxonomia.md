# Taxonomía en WordPress
- Categorías
	Clasifica el contenido. Base de la organización. Permite sub-categorías.
- Etiquetas
	Palabras clave. Keywords para SEO.
- Entradas
	Publicaciones, posts. 
- Páginas
	Contenido estático. Página de inicio, de contacto, etc.
