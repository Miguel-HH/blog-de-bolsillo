# ¿Cómo funciona la HERENCIA en JavaScript?
La palabra clave es **extends**.

````js
class constuctorHeredado extends constuctorOriginal {
	//
}
````

"constuctorHeredado" hereda todos los métodos de "constuctorOriginal".
