# Funciones como constantes en JavaScript

Las funciones anónimas nunca cambiarán en la ejecución de nuestro código, por lo cual es recomendado almacenarlas como constantes.

````js
const saludar = function (nombre){
	return `Hola ${nombre}`
}
````
