- Propiedades de modelo de caja (display, width, height, margin, etc.)
- Posicionamiento (position, top, right, bottom, left, etc.)
- Tipografía (text-transfrom, text-decoration, etc.)
- Decoración (background-image, color, etc.)
- Variables
- Mixins
