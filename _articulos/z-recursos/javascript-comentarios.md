# Comentarios en JavaScript

## Buenas prácticas

````js
/**
 * "Este es
 * un bloque
 * de código."
 */
````

### Atajos en VScode

- Convertir en comentario: `Ctrl+K` + `Ctrl+C`
- Revertir comentario: `Ctrl+K` + `Ctrl+U`
